## Quickstart (5 minutes)

```bash
npm install
npx patch-package
npm run web   # or: npm run android | npm run ios
```

## Contributing

Want to help strangers on the internet find love? There's three ways you can contribute!

1. Tell your friends about Bunk and share on social media! This is the best way to make it grow.
2. Raise a pull request. Developer instructions can be found at [DEVELOPER.md](DEVELOPER.md). See our [CONTRIBUTING.md](CONTRIBUTING.md) for PR tips and a checklist.
3. Support Bunk by using the app and providing feedback!
